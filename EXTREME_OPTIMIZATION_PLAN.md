## 🔴 优先级 1：Conv2D（1.04x → 5x+）

**当前问题**：
- 加速比太低（1.04x）
- 计算瓶颈（Compute 82.45%）

**优化策略**：
1. ✅ 使用 im2col 转换 + 优化 GEMM
2. ✅ 增加每个线程的工作量（处理多个输出）
3. ✅ 使用向量化加载（float4）
4. ✅ 优化内存访问模式

**预期提升**：1.04x → 5-8x

---

## 🟡 优先级 2：Reduction（2.25x → 5x+）

**当前问题**：
- 加速比较低（2.25x）
- 可以进一步优化 warp-level reduction

**优化策略**：
1. ✅ 使用 warp shuffle 替代共享内存
2. ✅ 增加每个线程的工作量
3. ✅ 优化 grid size
4. ✅ 使用向量化加载（float4）

**预期提升**：2.25x → 5-8x

---

## 🟡 优先级 3：Fused MoE（3.67x → 8x+）

**当前问题**：

**优化策略**：
1. ✅ 进一步优化向量化加载
2. ✅ 使用共享内存缓存输入

**预期提升**：3.67x → 8-10x
